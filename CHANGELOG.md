# Changelog

## 1.0.0 — 2026-07-12

ReceiptHubの最初の正式な業務フロー版です。

### Added

- 一般ユーザー / 管理者ユーザー / スーパーアカウントの権限分離
- スーパーアカウントによるユーザー権限変更
- ユーザー追加・削除・ステータス・パスワード管理
- サービスマスター、ユーザー別サービス、ユーザーによる利用登録・停止
- 月次領収書提出と従量課金/APIの「当月利用なし」申告
- 管理者手動実行によるOpenAI領収書ファイル名修正・検査
- 明確な不一致の自動取り下げと再提出依頼
- 曖昧な領収書の管理者レビュー用ハイライト
- カードご利用代金明細書のAI抽出、サービス・領収書照合、未提出行ハイライト
- 通常 / 重要リマインダーの対象判定
- 管理者編集可能な送信日、件名、本文
- Resend SMTPテスト送信と送信ログ
- 最大3ヶ月のファイル保存と期限切れ削除
- 初回ログインチュートリアル
- `VERSION` ファイルとバージョン付き配布フォルダー

### Changed

- アプリ名をReceiptHubへ統一
- 管理者の開始ページを提出履歴へ統一
- AI処理を定期Cronではなく管理者ボタンによる手動開始へ変更
- 同じサービスの複数カード明細行へ同一領収書を使い回さない照合方式へ変更

- `receipts.0014_emailreminderschedule_initial_body_template_and_more`
